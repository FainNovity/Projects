Rules to play super tic-tac-toe :


  1. Select any box from the ground on the left, you want to start the game with.
  2. click the mini-box on the ground on right, you want to fill in selected box.
  3. The mini-box you filled will be next selected box of your opponent.
  4. If a box is already completed/won by someone, player will get a chance to select the box.
  5. When a player wins a box, box will be marked as his sign(x or o).
  6. When a player has 3 consequencial wins in 3 boxes, he's declared as winner!!
